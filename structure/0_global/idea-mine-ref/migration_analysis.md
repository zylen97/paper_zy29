# Migration analysis — P02 / 徐进跨项目知识迁移

### P2: 多源知识融合

**方向 1**：制造业产线换型质量预测（半导体/新能源电池）
- 理论差异：分布时变性来源从地质变化转为设备老化/工艺参数漂移，TBME分块可与领域先验知识对比验证
- 假设变化：子域间差异最大化假设可能不成立，最优分块需与物理事件对齐

**方向 2**：供应链跨企业需求预测知识共享
- 理论差异：引入信息不对称与策略性行为新维度，多源融合权重受组织间信任调节
- 假设变化：源域数据可自由获取假设不成立，需嵌入联邦学习/隐私保护机制

**方向 3**：基础设施跨资产退化预测（桥梁群/风电场）
- 理论差异：时间尺度从秒-分钟拉伸到月-年，TBME从"去噪"转为"识别退化阶段"
- 假设变化：冻结特征提取器假设过于严格，需放宽为部分层微调或域自适应层
